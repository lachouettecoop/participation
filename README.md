# La Chouette Coop : Suivi de participation

Ce projet utilise [Next.js](https://nextjs.org/).

# Pré-requis

- [Node](https://nodejs.org/) v10.13+
- [Yarn](https://yarnpkg.com/)

# Configuration

Créez un fichier `.env` sur le modèle de `example.env`

# Développement

    yarn dev

# Production

    yarn build
