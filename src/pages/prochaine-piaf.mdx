# Prochaine PIAF

Cet indicateur permet d’informer sur l’état de la prochaine PIAF effectuée par
la personne. Cette information est récupérée depuis le
[Planning du LAB](https://docs.google.com/spreadsheets/d/1QryWNsR34pRuUfz5HyQPEIlTDa9JiSp1kUorcU9merM/edit#gid=1816688434).

**IMPORTANT : l’heure à laquelle les informations ont été récupérées est
indiquée en bas de page.** Les modifications ne sont pas répercutées en temps
réel.

L’indicateur affiche les informations suivantes :

- date de la prochaine PIAF à laquelle la personne est inscrite

Cette page présente les différents états possibles.

## Inscrit·e pour une PIAF

<ProchainePiaf
  date={format(addWeeks(new Date(), 2), "DD/MM/YYYY")}
  width={1 / 2}
  py={4}
/>

## Non inscrit·e pour une PIAF

<ProchainePiaf date="-" width={1 / 2} py={4} />

<ImportantText mt={4}>
  Action proposée : s’inscrire au planning pour une PIAF.
</ImportantText>
