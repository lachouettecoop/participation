# Récapitulatif Global

Cet indicateur permet d’informer sur l’état global de la participation effectuée
par la personne, en mettant en avant ce qui serait attendu.

Il affiche les informations suivantes :

- nombre de PIAF effectuées par la personne depuis son inscription
- nombre de PIAF attendues pour atteindre un équilibre (sur la base de **1 PIAF
  toutes les 4 semaines**)
- information sur le nombre de PIAF réalisées depuis la rentrée 2018
- l’adresse email utilisée pour établir cette concordance

Pour rappel : c’est l’adresse mail saisie sur le planning qui permet
d’identifier le Chouettos. Si l’adresse mail n’est pas renseignée ou bien
différente de celle donnée à l’inscription, alors l’outil ne fonctionne pas.

Cette page présente les différents états possibles.
